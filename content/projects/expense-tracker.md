## Problem

For my undergraduate final project, supervised by Dr. Fatemeh Sheikhi, I wanted
to build more than a simple expense CRUD app. The goal was a personal finance
system that helps users understand their financial behavior through tracking,
budgets, and transparent insights.

## Approach

The application uses React, Node.js, Express, MongoDB, and JWT authentication.
Users can manage income, expenses, categories, and budgets through protected
routes. The dashboard calculates monthly income, expenses, savings rate, top
spending categories, budget usage, and projected monthly spending.

## Implementation Notes

- Implemented JWT-based authentication and protected API routes.
- Designed MongoDB models for users, income, expenses, and budgets.
- Added category-level budget warnings for overspending and near-limit usage.
- Built Chart.js visualizations for transaction trends and category
  distribution.
- Designed a Persian RTL interface with Jalali-friendly date handling.
- Refactored the project structure for GitHub readiness with documentation,
  environment examples, and setup scripts.

## What I Learned

This project connected full-stack engineering with data-oriented product
thinking. The most important part was making the financial insights explainable
instead of presenting them as a black-box recommendation.
